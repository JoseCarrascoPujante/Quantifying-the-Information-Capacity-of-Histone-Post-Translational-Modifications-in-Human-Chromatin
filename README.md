# Reproducibility package: information capacity of histone PTMs

Companion code for the manuscript **"Quantifying the Information Capacity of Histone Post-Translational Modifications in Human Chromatin."** A single Jupyter notebook ([`reproducibility_package.ipynb`](reproducibility_package.ipynb)) regenerates every numerical result and Tables 1, 2, 3 and M1 in the article. All output is shown inline within the notebook, no file(s) are generated by it.

## Requirements

Python ≥ 3.10 with `pandas` ≥ 1.4 and `jinja2` ≥ 3.1.2 (tested with Python 3.13.14, pandas 3.0.1, jinja2 3.1.6). The notebook checks these versions on the first cell and stops with a clear message if they are not met.

## Run

The notebook is intended to run top to bottom. Runtime is a few seconds and no external data or network access is required.

## Reproducibility workflow overview

The notebook reproduces all numerical results in the manuscript from closed-form Shannon-entropy expressions, covering the full core-histone information hierarchy (theoretical capacity, accessible proteoform space, statistical occupancy, correlation correction and functional interpretability) and the parallel linker histone H1 hierarchy, together with the integrated core-plus-H1 estimates. Lower and upper bounds are propagated as ranges throughout. Values are computed in full precision and displayed at the manuscript's reporting precision (one or two decimals); tables render below each module and can be pasted directly and seamlessly into a word processor.

A single setup cell holds all parameters as the one source of truth, so editing it adapts the whole framework. The reproducibility modules follow the manuscript's reading order so that the notebook can be read and executed side by side with it. Each module asserts its computed results against the manuscript's published values; these checks are gated by `RUN_REGRESSION_CHECKS` and are skipped automatically when the parameters defined in the setup cell are changed for adaptation.

## Notebook index

| Reproducibility Module | Manuscript item |
|---|---|
| Module 1 | Table 1, theoretical capacity and accessible proteoform space (Results 2.1-2.2) |
| Module 2 | Table 2, core-histone PTM information hierarchy (Results 2.2-2.3) |
| Module 3 | Table 3, linker histone H1 and integrated estimates (Results 2.4) |
| Module 4 | Table M1, model parameters (Methods) |

The full description of each Reproducibility Module is available in **Supplementary Information S2** of the manuscript and in markdown cells above each module within the repproducibility_package.ipynb Jupyter notebook.
